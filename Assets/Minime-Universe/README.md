# Minime Universe

This folder contains the educational mini-games ecosystem that contributes resources to the main Mini-Me character.

## Structure

- **Core-Game/**: Main tamagotchi care systems
- **Side-Games/**: Educational mini-games that earn resources

## Design Philosophy

- Cross-game currency system
- Educational value in every game
- Resources earned benefit main character
- No stress mechanics - all games are designed for cozy, learning-focused gameplay
