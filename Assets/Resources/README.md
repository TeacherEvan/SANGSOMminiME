# Resources

Runtime-loadable assets for the Sangsom Mini-Me system.

## Structure

- **Outfits/**: Purchasable clothing items for character customization
- **Accessories/**: Hats, jewelry, and special items

## Asset Guidelines

- Keep textures optimized for mobile (max 1024x1024)
- Use shared materials where possible
- Follow naming convention: `category_itemname_variant`
